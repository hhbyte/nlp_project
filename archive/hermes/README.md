# Experiments

Notebook order:

- `00_pipeline_initial.ipynb` - first compact pipeline.
- `01_complete_pipeline_lab.ipynb` - full experiment notebook with the main trials.
- `02_model_ensemble_lab.ipynb` - ensemble comparison notebook.
- `03_preprocessing_and_cleaning.ipynb` - text cleaning and dataset preparation code.
- `04_matching_and_retrieval.ipynb` - exact matching, fuzzy matching, and retrieval trials.
- `05_classical_models_and_chapter_submission.ipynb` - TF-IDF, SVM, and chapter submission code.
- `06_optimization_rules_and_hybrid_submission.ipynb` - tuned rules and hybrid submissions.
- `07_embeddings_and_meta_ensemble.ipynb` - embedding and meta-ensemble experiments.
- `08_pipeline_entrypoints_and_package_notes.ipynb` - former CLI wrappers and package notes.
