# Archive

This folder keeps the exploratory work that led to the final report and best
submission. It is not the recommended entry point for reproduction; use
`best_submission/` for the final pipeline and `report/final_report/` for the
written report.

Contents:

- `daniel/` - Daniel's original notebooks, outputs, and best submission folder.
- `hermes/` - Hermes' trial-and-error notebooks, including archived versions of
  the former Python scripts/package code.
- `shanthosh/` - Shanthosh's exploratory notebooks and notes.
- `comparison_outputs/` - comparison CSVs from development.
- `hermes_outputs/` - generated predictions/submissions from Hermes' experiments.
- `models/` - model artifact notes/placeholders.

These files are preserved for transparency, but they may contain intermediate
paths, duplicated outputs, and experiments that were not part of the final
submitted system.
